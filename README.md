# Saghul's Tap

## How do I install these formulae?

`brew install saghul/tap/<formula>`

Or `brew tap saghul/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "saghul/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
